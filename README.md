# TimeWise AI

**TimeWise AI** is an intelligent workforce orchestration platform for industries with complex shift planning, staff coordination, compliance, and real-time operational disruption handling.

## Vision

TimeWise AI becomes the intelligent operating system for workforce management across healthcare, aged care, hospitality, manufacturing, security, logistics, and other industries that require mission-critical people planning.

## Features

- AI-powered scheduling optimization
- Demand forecasting and shortage prediction
- Skill, certification, location and availability matching
- Real-time replacement for unavailable staff
- Compliance-aware labor rule enforcement
- Integrations with HR, payroll, ERP, time-tracking, and workforce platforms
- Dashboard for workforce health, shift risk, and operational intelligence

## Project Structure

- `backend/`: Express API and orchestration services
- `frontend/`: React + Vite web UI

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   npm run install-all
   ```
2. Start development servers:
   ```bash
   npm run dev
   ```
3. Open the frontend at `http://localhost:5173`
